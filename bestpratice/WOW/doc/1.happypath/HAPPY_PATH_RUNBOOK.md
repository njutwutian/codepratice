# Happy Path 联调手册

## 启动
1. 在项目根目录执行 docker compose 启动。
2. 确认 mysql、adminer、goldenmount、northtoward 全部为 running。

## 验证步骤
1. 调用后端创建事件接口，返回成功。
2. 打开前端 events 页面，能看到新事件。
3. 进入详情页，字段显示完整。

## 成功标准
- 创建事件无报错。
- 列表与详情均能读取到同一条数据。
- 前后端接口路径一致，浏览器无跨域阻断。

## 常见故障速查
- 镜像拉取失败：检查镜像源是否为镜像站地址。
- 前端空列表：检查后端容器状态与 API 路径。
- CORS 错误：检查后端跨域配置是否生效。
